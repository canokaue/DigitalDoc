Como funciona


Cada documento recebido é verificado, criptografado e guardado na Blockchain IPFS.

O link de acesso é entregue para a entidade registradora diretamente em seu dashboard + pdf pronto para impressão.

O cliente também imediatamente recebe por SMS (graças à API da Twilio) o link de acesso + informações adicionais sobre seu documento. Este registro é perpétuo, portanto esqueça perdas, danos e invalidez.
